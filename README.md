
# Pick-Up Washu

 This is a sport events website for WashU students. 
## Website

https://pick-up-washu.vercel.app/

## Deployment

To run this project locally

First install all dependency
```bash
    npm install
```

In mainfoler
```bash
    npm run dev
```

you need to set up your own env variable
